# 개인정보 처리방침

[English](PRIVACY_POLICY.md)

**최종 업데이트: 2026년 2월 21일**

## 개요

ccpocket("앱")은 Claude와 Codex를 위한 모바일 원격 클라이언트입니다. 이 개인정보 처리방침은 앱이 어떤 데이터를 수집하고, 어떻게 사용하며, 사용자가 해당 데이터와 관련해 어떤 권리를 갖는지 설명합니다.

이 앱은 Anthropic 또는 OpenAI와 제휴, 보증 또는 관련 관계가 없습니다.

## 데이터 수집

### 수집하지 않는 데이터

- 분석 또는 텔레메트리
- 외부 서비스로 전송되는 크래시 리포트
- 개인 정보(이름, 이메일, 전화번호)
- 사용 추적 또는 행동 데이터
- 광고 식별자
- 위치 정보

### 기기에 로컬로 저장되는 데이터

다음 데이터는 사용자의 기기에만 저장되며, 개발자나 제3자에게 전송되지 않습니다.

- **앱 설정**: 테마, 언어, 알림 설정
- **연결 설정**: Bridge Server URL, 머신 구성
- **프롬프트 기록**: 로컬 데이터베이스에 저장되는 이전 프롬프트
- **채팅 초안**: 세션별 미전송 메시지 텍스트
- **자격 증명**: API 키와 SSH 자격 증명. 암호화된 보안 저장소에 저장됩니다

### 사용자의 Bridge Server로 전송되는 데이터

사용자가 직접 운영하는 Bridge Server에 연결하면, 앱은 WebSocket을 통해 다음 데이터를 주고받습니다.

- 채팅 메시지와 프롬프트
- 도구 승인/거부 응답
- 세션 관리 명령

이 통신은 사용자의 기기와 사용자가 소유한 서버 사이에서 직접 이루어집니다. 데이터는 개발자의 서버를 거치지 않습니다.

### Firebase Cloud Messaging(푸시 알림)

푸시 알림을 활성화하면 앱은 다음을 사용합니다.

- **Firebase 익명 인증**: 푸시 알림 전달을 위한 익명 식별자를 생성합니다. 이 식별자에는 개인 정보가 연결되지 않습니다.
- **Firebase Cloud Messaging(FCM)**: 코딩 에이전트가 승인을 요청하거나 작업을 완료했을 때 푸시 알림을 받기 위해 사용합니다.

푸시 알림은 앱 설정에서 언제든지 비활성화할 수 있습니다. Firebase는 전달 목적 외에 알림 내용을 저장하지 않습니다.

## 제3자 서비스

| 서비스 | 목적 | 공유 데이터 |
|--------|------|-------------|
| Firebase (Google) | 푸시 알림 전용 | 익명 기기 토큰 |

다른 제3자 서비스는 사용하지 않습니다.

## 데이터 보안

- 민감한 자격 증명(API 키, SSH 키)은 플랫폼 기본 암호화 저장소(iOS Keychain / Android Keystore)에 저장됩니다
- 모든 Bridge Server 통신은 Tailscale VPN 또는 로컬 네트워크로 보호할 수 있습니다
- 개발자가 관리하는 외부 서버에는 데이터가 저장되지 않습니다

## 아동의 개인정보

이 앱은 만 13세 미만 아동을 대상으로 하지 않으며, 아동의 데이터를 의도적으로 수집하지 않습니다.

## 사용자의 권리

ccpocket은 데이터를 사용자의 기기에만 저장하므로 다음을 할 수 있습니다.

- **데이터 삭제**: 앱을 제거하여 로컬에 저장된 모든 데이터를 삭제할 수 있습니다
- **데이터 확인**: 저장된 데이터는 앱 설정을 통해 확인할 수 있습니다
- **알림 비활성화**: 설정에서 푸시 알림을 끄면 Firebase 통신을 중단할 수 있습니다
