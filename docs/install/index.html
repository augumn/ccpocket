Quick start in 3 steps

1
Run Bridge

Run the Bridge Server on the Mac or Linux machine that has Codex or Claude installed.

npx @ccpocket/bridge@latest
2
Scan the terminal QR

Install the app, open CC Pocket, and scan the QR code printed by Bridge.

3
Start a session

Choose a project, pick Codex or Claude, and start working from your phone.

Need access away from your local network? Use Tailscale and connect to your machine's Tailscale address.
